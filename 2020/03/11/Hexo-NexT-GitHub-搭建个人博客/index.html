Hexo+NexT+GitHub 搭建个人博客

Posted on 2020-03-11 | In 工具
Words count in article: 1.3k
清晰完整地讲述如何使用Hexo,NexT,GitHub等工具，搭建自己的博客。减少麻烦，节省时间。

入门

  1. 安装 node.js。

  2. 安装 git。

  3. 任意磁盘位置，新建文件夹MyBlog。

  4. 安装Hexo。

    在MyBlog文件夹内，右键，Git Bash Here, 执行命令：

    npm install -g hexo-cli

    耐心等待一段时间，出现：

    然后执行命令：

    hexo init

    等待一段时间，出现：

    Hexo 安装完成。

  5. 安装NexT主题。

    执行命令：

    git clone https://github.com/iissnan/hexo-theme-next themes/next

    出现：

    修改/MyBlog/_config.yml文件。推荐使用Visual Sudio Code来编辑它。

    theme: landscape 修改为 theme: next

  6. 写好的文章放在哪里。

    MyBlog\source_posts 文件夹。如果你想发布一篇新文章，把它转为md文件放在这个文件夹中。现在这个文件夹中已经有一篇文章hello-world.md， 接下来把它发布到你的博客中。

  7. 连接GitHub与本地。

    创建一个GitHub 仓库，仓库的命名格式为：yourusername.github.io。

    修改/MyBlog/_config.yml文件, 找到deploy, 将其改为：

    deploy:
  type: git
  repo: https://github.com/yourusername/yourusername.github.io.git #注意修改yourusername
  branch: master

    执行命令：

    npm install hexo-deployer-git --save

    出现：

    依次执行命令：

    git config --global user.name "yourusername" #注意修改yourusername

git config --global user.email "youremail" #注意修改youremail

    然后，执行命令：

    hexo d

    如果提示输入GitHub用户名和密码，输入。

    恭喜，博客发布成功。前往https://yourusername.github.io，可看到你的博客。

进阶

  1. 文章的图片放在哪里

    强烈建议采用如下设置，以防文章图片无法正常显示。、

    修改/MyBlog/_config.yml文件:

    将post_asset_folder: false改为post_asset_folder: true

    然后，执行命令：

    npm install https://github.com/CodeFalling/hexo-asset-image -- save

    设置完成后。当执行 hexo new “文章名”命令 来创建文章的时候，会在/source/_posts目录下生成名字相同的md文件和文件夹。将你想在博文中插入的图片放到那个文件夹中。

    在md文件中引用图片的时候，使用如下代码即可：

    <img src=P70507-171306-EFFECTS.jpg width=500 title="图片名" align=center /> #设置了图片宽度，高度随之按比例变化
或
![](P70507-171306-EFFECTS.jpg "野菊") #图片原大小

  2. 选择NexT主题的风格

    修改/MyBlog/themes/next/_config.yml文件。

    • Muse风格

    • Mist风格

    • Pisces风格

    • Gemini风格（更立体，但布局与Pisces相同）

  3. 设置网站图标icon

    将你准备好的图标，放在\MyBlog\themes\next\source\images文件夹中，替换掉favicon-16x16-next.png和favicon-32x32-next.png。

    刷新网页后：

  4. 设置博客Menu

    编辑hexo/themes/next/_config.yml，我个人喜欢的配置如下：

    menu:
  home: /||home
  about: /about/||user
  tags: /tags/||tags
  #categories: /categories/||th
  archives: /archives/||archive
  #schedule: /schedule/||calendar
  #sitemap: /sitemap.xml||sitemap
  #commonweal: /404/||heartbeat

    依次执行命令：

    hexo new page about

hexo new page tags

hexo new page categories

      找到/blog/source/about/index.md文件，编辑它。我的格式如下：

    • Tags页设置

      找到/blog/source/tags/index.md文件

    • Categories页设置

      找到/blog/source/c*ategories/index.md*文件

    一篇文章中如何加入标签和分类呢？示范如下：

    ---
title: 示范tags和categories
date: 20xx-05-18 16:41:52
tags:
    - Hexo
    - 博客
categories:
    - 心得
    - 博客
---

  5. 博客头像

    将头像图片，例如avatar.gif，放置在 \MyBlog\themes\next\source\images 文件夹中

    编辑hexo/themes/next/_config.yml：

    #Sidebar Avatar
# in theme directory(source/images): /images/avatar.gif
# in site  directory(source/uploads): /uploads/avatar.gif
avatar: /images/avatar.gif

  6. 博客基本信息

    修改/MyBlog/_config.yml文件。将文件中，一些属性的值修改为你的个人信息。主要修改以下信息：

    title: 龙小歪
author: heloook
url: https://heloook.github.io

    社交链接设置。打开/MyBlog/themes/next/_config.yml文件，修改以下内容：

    social:
  #GitHub: https://github.com/yourname || github
  E-Mail: mailto:yourname@gmail.com || envelope
  #Google: https://plus.google.com/yourname || google
  #Twitter: https://twitter.com/yourname || twitter
  #FB Page: https://www.facebook.com/yourname || facebook
  #VK Group: https://vk.com/yourname || vk
  #StackOverflow: https://stackoverflow.com/yourname || stack-overflow
  #YouTube: https://youtube.com/yourname || youtube
  Instagram: https://instagram.com/yourname || instagram
  #Skype: skype:yourname?call|chat || skype

  7. 启用Disqus评论

    非常推荐使用Disqus评论系统。参考文章 Hexo开启disqus评论系统，进行设置。

优化

  1. 文章字数统计

    执行命令

    npm i --save hexo-wordcount

    打开hexo/themes/next/_config.yml文件，找到wordcount，属性值改为true

    post_wordcount:
  item_text: true
  wordcount: true
  min2read: false
  totalcount: false
  separated_meta: true

  2. 隐藏文章评论数

    打开 themes/next/source/css/_custom/custom.styl 文件，添加代码：

    //屏蔽标题下的评论数量
.post-comments-count {
  display: none;
}

  3. 版权声明

    打开hexo/themes/next/_config.yml文件，找到post_copyright，属性enable值改为true。

  4. 站内搜索等更多设置

    可以阅读文章Hexo 搭建个人博客系列：进阶设置篇
